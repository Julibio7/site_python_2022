Bem-vindo(a) à Segunda Edição do Workshop de Python para Dados de Microbiologia!

Um workshop online de introdução à programação com Python onde ensinamos como utilizar a linguagem para analisar e extrair informações relevantes dos seus dados microbiológicos!

O curso é totalmente gratuito! Por isso, faremos uma seleção dos candidatos.

Introdução à programação com Python

Introdução à Programação com Python para alunos da área de Microbiologia com pouca ou nenhuma experiência em programação. Contamos ainda com um material especialmente desenvolvido para o curso e os participantes envolvidos.
